# nvim-config
My personal nvim configuration. The code contains some comments to better understand some of the configuration options.

## Requirements

Here is a complete list of all the external requirements that are needed for this configuration:

- A [NerdFont](https://www.nerdfonts.com/) configured as your terminal font is required. I personally use Hack.
- For Lua LSP support, I have followed the following [instructions](https://luals.github.io/#neovim-install), building from scratch.
